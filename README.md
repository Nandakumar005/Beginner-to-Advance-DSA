# DSA Learning Journey

This repository is my personal Data Structures and Algorithms (DSA) learning journal.

The primary goal of this repository is to learn, document, and understand DSA concepts while building a structured set of notes for future revision.

Instead of simply storing solutions, I focus on understanding:

* Why an algorithm exists
* What problem it solves
* How it works internally
* Time and space complexity
* Step-by-step dry runs
* Python implementations
* Common beginner mistakes

This repository is written in a beginner-friendly style and documents my learning process as I progress through different DSA topics.

## Learning Method

For each topic, I generally follow this process:

1. Understand the problem
2. Learn the intuition behind the algorithm
3. Perform manual dry runs
4. Analyze time and space complexity
5. Implement the algorithm in Python
6. Document the concept in Markdown
7. Solve related problems

## Why Public?

I created this repository primarily for my own learning and revision.

However, since these notes are written in a beginner-friendly manner, anyone learning DSA is welcome to use them as a reference.
